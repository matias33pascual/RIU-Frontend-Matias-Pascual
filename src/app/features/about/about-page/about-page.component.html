Acerca del Proyecto

Aplicación web desarrollada con Angular 19 que permite gestionar un catálogo de superhéroes.

La aplicación incluye funcionalidades como:

  • 🦸‍♂️ Listado completo de superhéroes
  • 🔍 Búsqueda en tiempo real
  • ➕ Creación de nuevos superhéroes
  • ✏️ Edición de superhéroes existentes
  • 🗑️ Eliminación de superhéroes
  • 📱 Diseño responsive con Angular Material
  • ⚡ Server-Side Rendering (SSR) con Angular Universal

Tecnologías Utilizadas

Angular 19 TypeScript Angular Material RxJS Angular Universal (SSR) Signals Standalone Components
